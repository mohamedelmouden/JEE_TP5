------------------------------------"spring-hibernate-demo : "-----------------------------------------

Projet d'apprentissage — intégration Spring Framework + Hibernate ORM + MySQL.


-------------------------------------"Technologies :"---------------------------------------------

- Java JDK 8+

- Spring Framework (spring-context, spring-orm, spring-jdbc)

- Hibernate 5

- MySQL 8

- Maven

---------------------------------------------------"Utilisation :"-------------------------------------------------------

- Lancer TestHibernate.java pour vérifier la configuration Spring/Hibernate.

- Lancer Presentation2.java pour tester l'ajout d'un produit en base.



---------------------------------------------"Méthodes IDao<T> :"-----------------------------------------

- create(T o) — Ajouter un objet

- delete(T o) — Supprimer un objet

- update(T o) — Modifier un objet

- findById(int id) — Rechercher par ID

- findAll() — Lister tous les objets
